# Docs

robot arm motor specs
- left
    - pin: GP5
    - 0: reach
    - 90: back
    - drift: 0
- right
    - pin: GP6
    - 0: back
    - 90: reach
    - drift: + 80
- lift
    - pin: GP4
    - 60: pen down
    - 0: pen raise
    - drift: 0

robot arm dimensions
- a: 23.2
